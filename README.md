# SEQAX408DevOpsTechVagrant

## Getting started
- Clone the repo HelloHttp repo: `https://github.com/drines-uc/hello_http`
- CD into the repo
- run `vagrant up` to execute
- `vagrant ssh`
- `cd /hello_http`
- `gcc -o dummyserv dummy_serv.c`
- to run: `./dummyserv`
- `exit` when done

Notes:
-`vagrant init gyptazy/ubuntu22.04-arm64`
